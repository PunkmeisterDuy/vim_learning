# Navigation

## Basic Navigation
h j k l ; left down up right


gg ; top of page
G ; bottom of page

w ; next word
W ; next WORD
b ; previous word
B ; previous WORD
e; end of word
E; end of WORD

## Tabs
:tabnew ; new tab
:tabprevious ; :tabp ; gT ; previous tab
:tabnext ; :tabn ; gt ; next tab

:e ; edit
:w ; write
:q ; quit
:a ; all
combine with :wqa to write and quit all tabs

### File System
:Sexplore ; Opens file system horizontally
:Vexplore ; Opens file system vertically 
:Texplore ; Opens files system with text only
